# Reproduce round function from MS Excel

See:
https://stackoverflow.com/questions/59392686/r-how-to-define-round-function-like-in-excel

## Usage

``` r
fct_round(.num, .digits)
```

## Arguments

- .num:

  number to be rounded

- .digits:

  Number of digits to be passed to round()

## Value

a rounded numerical value
